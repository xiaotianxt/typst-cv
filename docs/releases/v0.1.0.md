# typst-cv v0.1.0

Initial public release of `typst-cv`.

## Highlights

- Modular Typst resume template with one file per work experience or project
- Multiple role-specific one-page resume profiles from a shared YAML dataset
- Compile-time module overrides for quick custom resume assembly
- Local validation workflow with `make check-all`
- GitHub Actions CI for compiling all profiles on push
- README preview assets and template-ready repository setup

## Chinese support

- Added `README.zh-CN.md` for Chinese-speaking users
- Added default Chinese serif font fallbacks for mixed English and Chinese content
- Documented how to customize CJK fonts locally

## Included sample profiles

- `software-engineer`
- `infra-platform`
- `ml-systems`
- `research-heavy`
